# Data Mining Assignment

## Yêu cầu
- Python 3.x

## Cài đặt

1. Clone repository (nếu chưa có trên máy):
```bash
git clone https://github.com/Nminnnnnnn/DataMining.git
cd DataMining
```

2. Cài đặt các thư viện cần thiết từ requirements.txt:

```bash
pip install -r requirements.txt
```

3. Chạy project
Chạy script chính từ thư mục gốc:

```bash
python -m src.main
